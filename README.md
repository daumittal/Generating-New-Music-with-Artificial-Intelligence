# Generating-New-Music-with-Artificial-Intelligence
In this project, we'll blend our technical prowess with creative flair to produce unique music using pretrained AI frameworks. Through Python programming, we'll develop interactive applications that enable users to input text descriptions (prompts) of musical styles and instruments, generating AI-composed music aligned with these prompts. Throughout the project, we'll explore various prompts to unravel how AI models interpret and generate music.

Our journey will encompass GUI programming and event handling as we construct an interactive user interface for music generation. This interface will seamlessly integrate text boxes and buttons, allowing users to interact with the AI music generation functions. Furthermore, we'll delve into the impact of prompts on the quality and diversity of AI-generated music, fostering an understanding of prompt engineering's influence on musical outcomes.

Lastly, we'll compare and analyze outputs generated from different prompts, providing insights into the nuanced effects of prompt selection on the AI music generation process.
